Project Overview

What It Is

This personal portfolio website is a fully responsive site designed and built from scratch using HTML and CSS. It serves as a central hub to showcase my background, technical skills, projects, and contact information to potential employers, collaborators, and peers.

What It Achieves

The site provides a polished, professional online presence that highlights my abilities as a Computer Science student. It features smooth scroll-reveal animations, a responsive navigation menu with a hamburger toggle for mobile users, a photo gallery with a lightbox viewer, and distinct sections for skills, projects, and contact details — all without any external JavaScript frameworks.

Technologies Used

HTML CSS

Key Features

  • Fully responsive layout that works on desktop, tablet, and mobile
  • Opening intro animation sequence
  • Sticky navigation bar with smooth scrolling
  • Intersection Observer–powered section reveal animations
  • Photo gallery with keyboard-accessible lightbox and touch swipe support
  • Clean, modern design using a consistent blue color palette
